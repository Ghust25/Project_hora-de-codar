Aula 10

Eventos

- Podemos ativar eventos nos componentes para dispararalgum método; - Um evento clássico que utilizamos muito é o click; - A sintaxe é: (click)="algumaFunção()" - Os métodos ficam na classe; - Futuramente utilizaremos este recurso para acessar uma API;

Aula 11

Emitindo eventos

- Podemos comunicar eventos de um componente filho para um componente pai; - Para isso vamos precisar do @Output, que é a saída do evento do componente filho; @Input do pai para o filho @Output do filho para o pai - Na tag de invocação do componente, escolhemos um método para executar quando o evento for emitido; - Exemplo: (emit)="onEmit()"

Aula 12

Renderização de listas

- Outro recurso importante é o loop em listas; - Para isso vamos precisar de uma propriedade com os itens da lista; - E no template do componente vamos utilizar o *ngFor; - A sintaxe é: *ngFor="let item of items" - Desta maneira podemos resnderizar template baseado em dados;
